Just ADOFAI Overlayer Working With ***DirectX11 Hooking.***

# Features
* Own Editor
  * Draggable Texts
  * Settings Window per Text
  * User-Friendly UIs
* Text system similar to Unity
  * Rich Text (currently, <color> only) Implementation
